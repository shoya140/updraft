Installation

0. Install require packages.

$ npm install -g bower grunt-cli coffee-script
$ gem install sass

1. Git clone or download updraft.

$ git clone git@github.com:Mrk1869/updraft.git

2. Install packages for grunt.

$ npm install

3. Download libraries and copy them to assets/ directory.

$ grunt bower:install

4. Start auto-compiling sass and coffee script.

$ grunt server
