Hello, updraft

The minimal scaffold for web applications.

Download

Necessary and Sufficient

HTML5 Boilerplate is a professional front-end template for building fast, robust, and adaptable web apps or sites.

Quick

Only 3 steps to scaffold a simple web app. Installing packages, installing libraries, and starting the web server. That's it.

Scalable

You might want to use tornado, sinatra or other web app framesetrows later. Yet no problem. All resources are in app/assets/ directory. Nothing to do to scale your app.

Installation

Require

Install require packages.

$ npm install -g bower grunt-cli coffee-script
$ gem install sass

How to start

1. Install packages for grunt.

$ npm install

2. Download libraries by using bower and copy them to assets/ directory.

$ grunt init

3. Start auto-compiling sass and coffee script.

$ grunt server
